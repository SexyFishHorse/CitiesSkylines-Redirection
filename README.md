## Installation

### Nuget feed details 

This library is available as a nuget package on the SexyFishHorse-CitiesSkylines nuget feed:

Description | URL
------------|----

Symbol server url | `https://www.myget.org/F/sexyfishhorse-citiesskylines/symbols/`
